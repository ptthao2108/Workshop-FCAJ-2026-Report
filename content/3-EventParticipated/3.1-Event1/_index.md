# Reflection Report: “AWS re:Invent Recap HCMC”

### Event Objectives

- Update key highlights from **AWS re:Invent**
- Explore trends in **infrastructure and application modernization** on AWS
- Learn about new advancements in **AI / ML / GenAI**
- Gain insights into how AWS supports businesses in **building, operating, and scaling systems**

### Main Content of the Event

The event was divided into two main tracks:

#### Track 1: Infrastructure & Application Modernization
- Infrastructure and application modernization
- **Hybrid Cloud & Multi-Cloud** strategies
- Simplifying containerized applications on AWS
- Optimizing workloads with appropriate database solutions
- Modernization with AI support
- System security and risk detection
- Announcement of **Vietnam DirectConnect POP**

#### Track 2: Analytics / ML / AI & Innovation
- New trends in **ML/AI**
- Vector Database on S3
- Natural language querying with OpenSearch
- Applying **Nova** and **Bedrock** for GenAI
- Working with AWS resources via **MCP**
- Multimodal Retrieval for Bedrock Knowledge Bases
- **MLOps** and model training with **Amazon SageMaker AI**

### Key Highlights

#### 1. Infrastructure Modernization & Networking

- **Previous challenges**: Network connectivity between regions (VPCs) was complex, difficult to scale, and prone to failure if a single Availability Zone (AZ) encountered issues.

- **New solutions**:
   + Use **Cloud WAN and Transit Gateway** for easier large-scale network management  
   + Introduction of **Regional NAT Gateway**: Improves system stability by removing dependency on a single AZ, enhancing availability and cost efficiency  
   + Build more flexible systems with **Hybrid Cloud** and **Multi-Cloud** models  
   + **Vietnam DirectConnect POP**: Direct connection point that reduces latency for the Vietnamese market  

#### 2. Generative AI (GenAI)

- **Nova & Bedrock Models**: Introduced a new generation of AI models (Nova) capable of multimodal processing (text, image, video) with faster response times  

- **Model Context Protocol (MCP)**: A new standard that enables AI agents to connect more intelligently with data sources and external tools  

- **AI-Powered Modernization (Kiro)**: A tool that leverages AI to automatically analyze and recommend security policies, reducing manual configuration efforts  

#### 3. Developer Tools & MLOps

- **Amazon SageMaker AI**: Enhancements that make training and deploying AI models more automated and cost-efficient  

- **Vector Database on S3**: A cost-effective yet high-performance approach to storing and querying AI-related data  

### Key Takeaways

- **Agentic AI mindset**: AI is no longer just for Q&A—it can autonomously perform tasks (e.g., browser automation, API calls) to assist users  
- **Infrastructure flexibility**: Better understanding of **multi-cloud integration** between AWS, Azure, and Google Cloud  
- **Operational optimization**: Awareness of tools that help automate repetitive system management tasks

The content was diverse, offering a high-level understanding of multiple emerging topics and highlighting key AWS services currently gaining traction.
